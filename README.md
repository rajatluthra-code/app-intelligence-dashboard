# App Intelligence Dashboard

Interactive daily performance dashboard for App Intelligence — tracking QBO Web & TurboTax Online performance tickets.

## Features

- **Multi-day data**: Date picker to navigate between daily reports
- **7 tabs**: Overview, Closures, Past Due, Feedback, Workload, Trends, Actions
- **14+ interactive charts**: Powered by Chart.js with drill-down capabilities
- **Historical trends**: Automatic trend analysis across available report dates
- **Slide-out detail panels**: Click KPIs for drill-down views
- **Responsive design**: Works on desktop and mobile

## How It Works

Each day's report data is stored as a JSON object in the `DAILY_DATA` structure inside `index.html`. The daily report agent appends new entries automatically.

## Data Source

- JIRA tickets created by App Intelligence Service Bot
- Smartsheet tracking for feedback and blocker responses
- Auto-generated by AI Agent
